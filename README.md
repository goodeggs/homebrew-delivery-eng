homebrew-devops
---------------
The DevOps Team's Homebrew formulae.

* `nvmish` -- an rvm-like node version selector that reads engines.node from package.json
* `ranch` -- the Ranch CLI
* `goodeggs` -- the Good Eggs CLI
* `devops` -- the DevOps CLI

----

## Installation

```
brew install goodeggs/devops/nvmish
```
